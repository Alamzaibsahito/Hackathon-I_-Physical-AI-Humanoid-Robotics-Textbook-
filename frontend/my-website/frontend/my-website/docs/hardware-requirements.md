To effectively participate in the course and complete hands-on exercises, the following hardware specifications are recommended:

| Component | Minimum | Recommended |
| :-------- | :------ | :---------- |
| CPU | Intel Core i5 (8th Gen) or AMD Ryzen 5 2600 | Intel Core i7 (10th Gen) or AMD Ryzen 7 3700X |
| RAM | 16 GB DDR4 | 32 GB DDR4 |
| GPU | NVIDIA GeForce GTX 1060 (6GB) | NVIDIA GeForce RTX 3060 (12GB) or higher |
| Storage | 256 GB SSD | 1 TB NVMe SSD |
| Operating System | Ubuntu 20.04 LTS (64-bit) | Ubuntu 22.04 LTS (64-bit) |
| Internet Connection | Broadband | High-speed broadband |

### Optional Hardware for Advanced Projects

| Component | Minimum | Recommended |
| :-------- | :------ | :---------- |
| Robotic Arm | Robotis OpenManipulator-X, UR3e | Humanoid Robot Kit (e.g., Robotis OP3, Unitree Go1) |
| Sensors | Basic webcam, IMU (e.g., MPU-6050) | High-resolution stereo cameras, LiDAR (e.g., RPLIDAR S1), Force/Torque sensors |
| Microcontrollers | Arduino Uno, Raspberry Pi 4 | NVIDIA Jetson Nano, NVIDIA Jetson Xavier NX |

*Note: The optional hardware is recommended for students who wish to undertake advanced physical robotics projects. The core curriculum can be completed with simulation environments.*
