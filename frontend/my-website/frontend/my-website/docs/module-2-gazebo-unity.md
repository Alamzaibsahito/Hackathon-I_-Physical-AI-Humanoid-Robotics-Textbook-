### Week 7-9: Gazebo Simulation

*   **Topics:** Gazebo environment setup, creating custom worlds, integrating ROS2 with Gazebo, sensor simulation (LiDAR, cameras), physics engines.
*   **Hands-on:** Designing a custom Gazebo world, simulating a robot with various sensors, data acquisition from simulated sensors.

### Week 10-11: Unity Robotics Simulation

*   **Topics:** Unity basics for robotics, creating robotic assets, Unity Robotics Hub, integrating ROS2 with Unity, advanced visualization.
*   **Hands-on:** Building a simple robot in Unity, connecting to ROS2 for control, real-time data visualization.
