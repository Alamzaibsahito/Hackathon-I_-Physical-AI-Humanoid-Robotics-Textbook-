### Week 1-3: Introduction to ROS2

*   **Topics:** ROS2 architecture, nodes, topics, services, actions, ROS2 workspaces, basic command-line tools.
*   **Hands-on:** Setting up ROS2 environment, creating custom ROS2 packages, publishing/subscribing to topics, controlling a simulated robot.

### Week 4-6: Advanced ROS2 and Robot Kinematics

*   **Topics:** URDF (Unified Robot Description Format), robot kinematics (forward and inverse), ROS2 navigation stack, SLAM (Simultaneous Localization and Mapping).
*   **Hands-on:** Building a URDF model, implementing forward kinematics, navigating a robot in a simulated environment using ROS2 navigation.
